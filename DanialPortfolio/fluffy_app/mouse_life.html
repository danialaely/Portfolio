Casual Game

Mouse Life

A stealth-based escape game where you guide a clever mouse to collect cheese, rescue a captured mouse, and return safely to the mouse house—all while avoiding a patrolling cat.

Fully developed the game including:

  • level design
  • progression system, and AI behavior.
  • Built multiple weapons to trap or slow the cat.
  • Integrated a complete shop and currency system using PlayFab.
  • Monetized through Unity Ads and IAPs.
  • Designed and balanced multiple levels with increasing difficulty.

Engine: Unity | Language: C# | Visual Studio | PlayFab, UnityAds

Platform: Android (Google Play)
