Casual Game

Cube Dash

A fast-paced endless runner where players swipe to change the shape of a cube to match incoming obstacles. The objective is to react quickly and survive as long as possible through increasingly difficult levels.

Fully developed the game including:

  • Developed complete gameplay mechanics including shape-shifting, obstacle spawning, speed scaling, and collision handling.
  • Focused on creating responsive controls and smooth performance to deliver an addictive hypercasual experience.

Engine: Unity | Language: C# | Visual Studio

Platform: Android (Google Play)
